# 🇪🇹 National Ethio Cyber Talent Weekend Program

Welcome to the official repository for the National Ethio Cyber Talent Weekend Program! This platform is designed to track your progress, practice real-world Git workflows, and build your technical skills in software development and cybersecurity.

---

## 📌 How to Submit Your Tasks

Please follow these exact steps to submit your work using standard GitHub practices (Forking and Pull Requests):

### 1. Fork this Repository
Click the **Fork** button at the top-right of this page to create a copy of this repository under your own GitHub account.

### 2. Clone Your Forked Repository
Open your terminal (Kali Linux/Ubuntu) and run:
```bash
git clone [https://github.com/YOUR_USERNAME/National-Ethio-Cyber-Talent-weekend-program-.git](https://github.com/YOUR_USERNAME/National-Ethio-Cyber-Talent-weekend-program-.git)
cd National-Ethio-Cyber-Talent-weekend-program-
(Replace YOUR_USERNAME with your actual GitHub username).

3. Create a New Branch
Do not work on the main branch. Create a new branch named after your task and your name:
# Format: task-number/your-name
git checkout -b task-01/natnael

4. Work on Your Task
Add your code, write-ups, or scripts inside the Submissions/[your-name]/ folder.
5. Commit and Push
Once you are done, save your changes and push them to your GitHub account:
Bash


git add .
git commit -m "Submit Task 01 - Natnael"
git push origin task-01/natnael

6. Open a Pull Request (PR)
Go to your forked repository on GitHub.
Click the "Compare & pull request" button.
Add a clear title (e.g., Task 01 Submission - Natnael) and click Create pull request.
Wait for the mentor's review! Feedback will be given directly on your code.

Happy Hacking & Coding! 💻🛡️
